# StarBerry Brand Guidelines

This skill provides comprehensive brand guidelines for StarBerry, including logo usage, color palettes, typography, and design standards.

## Logo System

StarBerry uses a three-tier logo system. Always select the appropriate mark based on the context and available space.

### Primary Mark

The Primary Mark is the main StarBerry logo featuring the star icon above the wordmark.

**Use for:**
- Office entrance signage
- App splash/load screens
- Document covers
- Business card covers
- Slide presentation covers
- Phone case covers

**Placement:**
- Vertically-symmetrical design
- Place centrally with additional content "horizontal align center" below
- Minimum clearspace: 2 × B on all sides (left, right, above, below)

**Minimum scale:**
- 70px / 25mm for digital/print

**Available files:** See assets/logos/primary/ for PNG files in Positive, Reverse, Grayscale, and 1-Colour variations

### Secondary Mark

The Secondary Mark features the star icon with wordmark to the right.

**Use for:**
- Website mastheads
- Email signatures
- Business card backs
- PowerPoint template headers
- Wristbands
- Bicycle frame wraps

**Placement:**
- Not vertically-symmetrical (use when impractical for centered layout)
- Allows horizontal layout for limited vertical space
- Left-align content below it
- Minimum clearspace: 1 × B above and below, 2 × B on left and right

**Minimum scale:**
- 100px / 35mm for digital/print

**Available files:** See assets/logos/secondary/ for PNG files in Positive, Reverse, Grayscale, and 1-Colour variations

### Tertiary Mark

The Tertiary Mark is the star icon only (no wordmark).

**Use for:**
- General office signage
- T-shirts
- Office mugs
- Animated masks within games or video
- Printed material emboss details
- Cappuccino stencils (yes, really!)

**Usage notes:**
- Should ideally be used where StarBerry brand name has been pre-established
- Can be used at very small scales or where the brand may be rendered poorly
- Can be used as an intriguing mark on its own
- Minimum clearspace: Half the height or width of the central shape (not including leaf element)

**Minimum scale:**
- 30px / 10mm for digital/print

**Available files:** See assets/logos/tertiary/ for PNG files in Positive, Reverse, Grayscale, and 1-Colour variations

### Legal Mark - StarBerry Games

Full mark: "StarBerry Games" with logo

**Important:** This mark is for legal use only and should NOT be used without specific approval.

## Capitalization Rules

**Critical branding requirement:**

- When referencing StarBerry in text, ALWAYS write with capitalized S and B: **StarBerry**
- The word "StarBerry" combines "Star" and "Strawberry" — a deliberate capitalization using UpperCamelCase
- Within unseparated character sets, use capitalized S and B

**Exceptions:**
- Entire titles or paragraphs set in ALL CAPS

## Web URL Standards

**Official URL:** StarBerry.games

**Presentation guidelines:**
- Use prefix 'http://www.' only when needed for clarity
- Distinguished from surrounding text by different font weight or color (preferably **StarBerry.games**)
- Avoid capitalizing the 'g' in .games

## Typography

### Primary Font: Source Sans Pro

StarBerry uses the **Source Sans Pro** font family for its modern, clean, sans-serif display.

**Approved weights:**
- Source Sans Pro Black
- Source Sans Pro Black Italic
- Source Sans Pro Bold
- Source Sans Pro Regular
- Source Sans Pro Italic

**Important restrictions:**
- **DO NOT use Source Sans Pro Black lower case** - it's too heavy
- **DO NOT use Source Sans Pro Black Italic lower case** - it's too heavy
- Only the five weights listed above are recommended for StarBerry branding

### Fallback Font: Arial

When Source Sans Pro is unavailable, substitute with Arial in these weights:
- Arial Black
- Arial Bold
- Arial Regular
- Arial Italic

## Logo Alignment Notes

### Primary Mark
- Vertically-symmetrical
- Place centrally in design applications
- Additional content should be "horizontal align center" below

### Secondary Mark
- Not vertically-symmetrical
- Left-aligned content placement
- Use when centered layout is impractical

### Tertiary Mark
- Slightly asymmetrical with leaf element overhanging on the right
- Ensure alignment based on the central shape

### Text Alongside Secondary Mark
- Match text precisely to the StarBerry wordmark's 'x' height
- Follow clearspace guidance (BB on sides)
- SSP Black should never be used alongside to ensure StarBerry wordmark retains prominence

## Brand Usage Guidelines

### DO:
- Use logos at or above minimum scales
- Maintain proper clearspace around all marks
- Use appropriate color palette for background
- Write "StarBerry" with capital S and B
- Use StarBerry.games (lowercase 'g')
- Apply proper typography hierarchy
- Select the correct logo file variation for your background

### DO NOT:
- Recolor any logo element
- Stretch, rotate, shear, or distort any logo element
- Place any element on busy backgrounds
- Place any element on alternative background colors
- Replace, retype, or refactor the StarBerry wordmark text
- Add drop shadow, strokes, or other FX to any logo element
- Cut or separate brand elements
- Merge the ribbon device segments (except for animated masking purposes)

## Minimum Scales

To ensure reproduction of the StarBerry marks remains identifiable, always use logos above these minimum sizes:

- **Primary Mark**: 70px / 25mm
- **Secondary Mark**: 100px / 35mm
- **Tertiary Mark**: 30px / 10mm

If a mark or device cannot be used above minimum sizes, replace it with text representation of "StarBerry".
