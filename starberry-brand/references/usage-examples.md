# StarBerry Brand Usage Examples

## Logo Usage Examples

### Correct Usage Examples

#### Primary Mark - Centered Composition
```
[Logo centered]
StarBerry
[Content centered below with proper clearspace]
```

**Scenario**: Company presentation cover
- Primary mark placed centrally at top
- All supporting content horizontally centered below
- 2 × B clearspace maintained on all sides

#### Secondary Mark - Header Layout
```
[Logo + StarBerry text] [horizontal layout]
Left-aligned content follows below
```

**Scenario**: Website header, email template
- Secondary mark aligned to left
- Content flows naturally below in left-aligned format
- 2 × B clearspace on left and right, 1 × B above and below

#### Tertiary Mark - Accent Usage
```
[Small star icon used as decorative element]
```

**Scenario**: T-shirt design, coffee mug, social media profile picture
- Star icon used where brand is already established
- Minimum 30px/10mm maintained
- Clearspace: half the central shape's height/width

### Common Usage Scenarios

#### 1. Business Card
**Front**: Primary mark centered
**Back**: Secondary mark in header, contact details left-aligned

#### 2. PowerPoint Presentation
**Cover slide**: Primary mark centered with title centered below
**Content slides**: Secondary mark in header, content left-aligned

#### 3. Email Signature
**Layout**: Secondary mark with contact information to the right
**Size**: 100px minimum width

#### 4. Website
**Header**: Secondary mark, far left
**Favicon**: Tertiary mark, 16×16px minimum

#### 5. Merchandise
**T-shirt**: Tertiary mark centered on chest
**Coffee mug**: Tertiary mark or Primary mark wrapped around
**Tote bag**: Primary or Secondary mark depending on layout

## Incorrect Usage Examples

### ❌ DO NOT: Recolor Logo Elements
```
[Star logo in pink/different colors]
```
**Why**: Breaks brand consistency, confuses identity

### ❌ DO NOT: Stretch or Distort
```
[Logo stretched horizontally or vertically]
```
**Why**: Distorts brand perception, looks unprofessional

### ❌ DO NOT: Place on Busy Backgrounds
```
[Logo over complex pattern or photo]
```
**Why**: Reduces legibility, weakens brand impact

### ❌ DO NOT: Add Effects
```
[Logo with drop shadow, glow, or outline]
```
**Why**: Clutters design, inconsistent with clean brand aesthetic

### ❌ DO NOT: Retype Wordmark
```
[Hand-typed "StarBerry" text replacing official wordmark]
```
**Why**: Font/spacing inconsistencies, diminishes professional appearance

### ❌ DO NOT: Separate Logo Elements
```
[Star icon and wordmark placed in different locations]
```
**Why**: Breaks logo cohesion, except for Tertiary mark which is designed standalone

### ❌ DO NOT: Use Below Minimum Size
```
[Tiny logo that's illegible]
```
**Why**: Brand becomes unidentifiable; use text "StarBerry" instead

### ❌ DO NOT: Place on Wrong Color Background
```
[Positive palette logo on dark background]
[Reverse palette logo on light background]
```
**Why**: Poor contrast, looks washed out or too heavy

## Brand Voice and Messaging

While primarily a visual identity guide, here are key messaging principles:

### Tone
- **Friendly yet professional**: Approachable but competent
- **Clear and direct**: No jargon unless necessary
- **Optimistic**: Positive framing, solutions-focused
- **Playful but not childish**: Fun without sacrificing credibility

### Writing Style
- Always write "StarBerry" with capital S and B
- Use active voice
- Keep sentences concise
- Write for clarity first

### Messaging Priorities
1. Innovation and creativity
2. Quality and reliability
3. User-centric approach
4. Fun and engaging experiences

## Application-Specific Guidelines

### Social Media
- **Profile Picture**: Tertiary mark or Primary mark (depending on platform crop)
- **Cover Photo**: Large Primary or Secondary mark with brand colors
- **Post Graphics**: Use approved color palettes, maintain visual consistency

### Packaging
- **Primary surface**: Primary or Secondary mark prominently displayed
- **Background**: Solid brand color or white
- **Typography**: Follow hierarchy guidelines
- **Minimum size**: Ensure logo meets minimum scales even on small packages

### Digital Products (Apps/Games)
- **App Icon**: Tertiary mark or simplified Primary mark
- **Splash Screen**: Primary mark centered on brand color background
- **In-game**: Tertiary mark for small branding moments
- **UI Elements**: Use brand colors for interactive elements

### Print Advertising
- **Headline**: Large, impactful typography from approved weights
- **Logo placement**: Upper third or lower right, maintaining clearspace
- **Color usage**: Full color when possible, solid Rhino or black for limited printing
- **Photography/Illustration**: Should not compete with logo; maintain clearspace

### Office/Environmental
- **Signage**: Primary mark for main entrance, Tertiary for directional
- **Business Cards**: Follow templates (Primary front, Secondary back)
- **Letterhead**: Secondary mark in header
- **Presentation Templates**: Follow slide guidelines above

## Quality Control Checklist

Before finalizing any StarBerry branded material, verify:

- [ ] Correct logo version selected (Primary/Secondary/Tertiary)
- [ ] Logo meets minimum size requirements
- [ ] Proper clearspace maintained around logo
- [ ] Appropriate color palette for background (Positive/Reverse)
- [ ] Colors match exact hex/Pantone specifications
- [ ] Typography uses approved Source Sans Pro weights
- [ ] No prohibited effects or distortions applied
- [ ] "StarBerry" written with capital S and B
- [ ] Overall design aligns with brand aesthetic
- [ ] Files saved in appropriate formats and resolutions
