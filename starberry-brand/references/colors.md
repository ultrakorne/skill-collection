StarBerry uses two distinct color palettes optimized for different background contexts. Always select the appropriate palette based on the background color of your design.

## Color Naming Convention

Color names are provided for identification during brand communication. The creative names (Coral, Sushi, Valencia, etc.) help distinguish colors in discussions without needing to reference hex codes.

## Pantone Identification

*Pantone colors are identified using CMYK values at pantone.com. For physical printing and production, always reference the official Pantone codes to ensure color accuracy.

## Usage Guidelines

### DO:
- Use colors directly from the specified palettes
- Match palette to background (Positive for light, Reverse for dark)
- Maintain color contrast for accessibility
- Use Rhino/Cloud Burst for primary brand elements

### DO NOT:
- Create custom color variations
- Mix Positive and Reverse palette colors inappropriately
- Alter opacity or add gradients unless specifically designed
- Use colors from outside these palettes for StarBerry branding

## Accessibility Notes

When using these colors for text and UI elements, ensure sufficient contrast ratios:
- Large text (18pt+): Minimum 3:1 contrast ratio
- Normal text: Minimum 4.5:1 contrast ratio
- UI elements: Minimum 3:1 contrast ratio

Always test color combinations for accessibility compliance.
