# StarBerry Typography Guidelines

## Primary Font Family: Source Sans Pro

Source Sans Pro is StarBerry's primary typeface, chosen for its modern, clean, sans-serif aesthetic that complements the brand's approachable yet professional identity.

### Approved Weights and Usage

#### Source Sans Pro Black
- **Use for**: Large titles, all-caps headings
- **Characteristics**: Heaviest weight, maximum impact
- **Restriction**: **DO NOT use in lower case** - too heavy and reduces readability
- **Recommended**: ALL CAPS only

#### Source Sans Pro Black Italic
- **Use for**: Stylized headings, emphasis in large text
- **Characteristics**: Heavy weight with italic slant
- **Restriction**: **DO NOT use in lower case** - too heavy and reduces readability  
- **Recommended**: ALL CAPS only

#### Source Sans Pro Bold
- **Use for**: Headings, subheadings, emphasis
- **Characteristics**: Strong weight, excellent for hierarchy
- **Case**: Both upper and lower case acceptable
- **Most common use**: Standard headings at all sizes

#### Source Sans Pro Regular
- **Use for**: Body text, paragraphs, general content
- **Characteristics**: Clean, readable, versatile
- **Case**: Both upper and lower case
- **Most common use**: All body copy and standard text

#### Source Sans Pro Italic
- **Use for**: Emphasis within body text, captions, quotes
- **Characteristics**: Gentle slant, maintains readability
- **Case**: Both upper and lower case
- **Most common use**: Subtle emphasis and stylistic variation

### Font Weights NOT Approved

The following Source Sans Pro weights exist but are **NOT recommended** for StarBerry branding:
- Source Sans Pro Extra Light
- Source Sans Pro Light
- Source Sans Pro Semibold

## Fallback Font: Arial

When Source Sans Pro is unavailable (e.g., in environments where custom fonts cannot be loaded), substitute with Arial.

### Arial Fallback Mapping

- **Source Sans Pro Black** → Arial Black
- **Source Sans Pro Black Italic** → Arial Black (no italic available, use regular Arial Black)
- **Source Sans Pro Bold** → Arial Bold
- **Source Sans Pro Regular** → Arial Regular
- **Source Sans Pro Italic** → Arial Italic

## Typography Best Practices

### Hierarchy
- Create clear visual hierarchy using size, weight, and color
- Titles should be significantly larger than body text
- Use consistent sizing within document types

### Leading (Line Spacing)
- Title text: Tight leading (90-100% of font size) for impact
- Body text: Comfortable leading (130-140% of font size) for readability
- Adjust based on line length and content density

### Tracking (Letter Spacing)
- Use default tracking for most applications
- Slight negative tracking acceptable for large ALL CAPS titles
- Never add excessive tracking that compromises readability

### Color Selection
- Always pair typography color with appropriate background palette
- Maintain sufficient contrast for accessibility
- Use brand colors from the approved palette only

### Case Usage
- ALL CAPS: Large titles, headings, labels (Source Sans Pro Black/Black Italic)
- Sentence Case: Subtitles, body text, most content
- Avoid mixing cases randomly within a single text element

## Digital vs Print Considerations

### Digital (Screen)
- Sizes shown are proportional; scale appropriately for screen resolutions
- Ensure text remains readable at actual pixel sizes
- Test on multiple devices and screen sizes

### Print
- Convert pixel/point sizes to appropriate print dimensions (mm)
- Consider paper quality and printing process
- Maintain minimum readable sizes for production method

## Accessibility Compliance

- Minimum body text size: 12pt for print, 16px for digital
- Ensure contrast ratios meet WCAG AA standards (4.5:1 for normal text)
- Avoid light weights for small text
- Test with screen readers when applicable

## Common Typography Mistakes to Avoid

### DO NOT:
- Use Source Sans Pro Black or Black Italic in lower case
- Mix too many font weights in a single design
- Use font weights not listed in approved weights
- Set body text in ALL CAPS (reduces readability)
- Use colored backgrounds that reduce text contrast
- Stretch or condense the typeface
- Use decorative effects (outlines, shadows, etc.) unless specifically designed

### DO:
- Maintain consistent hierarchy throughout documents
- Use appropriate leading for readability
- Select colors from the approved brand palette
- Test for readability at actual output size
- Maintain accessibility standards for all text
